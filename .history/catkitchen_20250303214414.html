CATS KITCHEN

esign Thinking – Design Basses Fidélité – Design Haute Fidélité – Responsive Web Design

Cats Kitchen est un site web responsive conçu pour répondre aux amoureux de chats qui souhaitent commander de la bonne nourriture saine pour leur chat.


  • PROJET : Site web responsive
  • RÔLE : UI & UX Designer
  • DATE : Janvier 2023 – Avril 2023
  • DURÉE : 3 mois

  • RESPONSABILITÉ :

    • Tests utilisateurs

WIREFRAME

J’ai inclus des considérations pour d’autres tailles d’écran en me basant sur mes wireframes précédents. Comme les utilisateurs font leurs achats à partir de différents appareils, j’ai pensé qu’il était important d’optimiser l’expérience de navigation pour différentes tailles d’appareils afin que les utilisateurs aient l’expérience la plus agréable possible.
